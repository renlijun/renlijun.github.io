# 猴子选大王

题目描述

给定一群猴子的数量，给定一个数到第几只淘汰，最后剩下的一个是大王。

LeetCode

约瑟夫问题

# 思路

# 移动指针

  • 每数一只猴子，如果不是大王
  • 当前猴子都追到末尾
  • 并且删掉当前猴子到位置
  • 如果是猴王，删除
  • 删除猴王以后，还需要按照之前的顺序接着数

# PHP

  • 方案：移动指针

    function elect (int $n, int $k) {

    $nums = range(1, $n);
    $i = 0;
    while (count($nums)>1) {
        //当前是要踢出的那只猴子
        if (($i+1) % $k == 0 ) {
            unset($nums[$i]);
        }
        //非猴王，把猴子追加到末尾 
        else {
            array_push($nums, $nums[$i]);
            unset($nums[$i]);
        }
        $i++; 
    }
    return $nums;
}
