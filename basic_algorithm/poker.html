# 判断是不是顺子

题目描述

随机发5张扑克牌，判断是不是顺子

  • 大小王为万能牌。

# 思路

  • 首选排序
  • 判断5张牌中，是否有重复的，有重复的肯定不是顺子
  • 过滤掉为0的牌
  • 用最大的牌 - 最小的牌，如果结果大于4, 不是顺子

# PHP

function check(array $poker)
{
    //排序
    sort($poker);

    $sortPoker = $poker;

    //去除0，如果非0的出现重复的，则证明不是顺子。
    $appearNum = [];
    foreach($sortPoker as $key=>$val) {
        if ($val == 0) {
            unset($sortPoker[$key]);
            continue;
        }

        if (empty($appearNum[$val])) {
            $appearNum[$val] = 1;
        } else {
            return false;
        }
    }

    //如果最大值减去最小的，大于4，则不是顺子
    $sortPoker = array_values($sortPoker);
    $min = $sortPoker[0];
    $max = $sortPoker[count($sortPoker)-1];

    if ($max-$min > 4) {
        return false;
    } else {
        return true;
    }
}
