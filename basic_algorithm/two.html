# 两数之和

题目描述

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

target = 6
nums = [1,3,5,6]
1
2

# PHP

第一种思路

第一种解法

暴力解决，双循环

复杂度 O(n^2)

function search2(array $nums, int $target)
{
    //复杂度为 O(n^2)

    $length = count($nums);
    for ($i=0; $i<$length; $i++) {
        for ($j=$i+1;$j<$length;$j++) {
            if ($nums[$i] + $nums[$j] == $target) {
                return [$i,$j];
            }
        }
    }
    return [];
}

第二种解法

y = target - x ，如果y存在于数组内，证明找到两数之和。 利用第三个变量数组，把值作为下标，这样来判断y是否在数组中

复杂度 O(n)

function search1(array $nums, int $target)
{
    //复杂度为 O(n)

    $reverseNums = [];
    foreach ($nums as $key=>$val) {
        if (isset($reverseNums[$target-$val])) {
            return [$key, $reverseNums[$target-$val]];
        } else {
            $reverseNums[$val] = $key;
        }
    }
    return [];
}
