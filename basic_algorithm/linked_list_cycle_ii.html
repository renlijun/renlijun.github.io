# 环形链表 II

题目描述

给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。

LeetCode

# 思路

# 记录指针

  • 记录指针轨迹到一个存储空间当中，如果存在，有环
    • 把每次走过的指针记录在一个空间当中
    • 判断当前针织是否存在于空间当中，查询复杂度为O(1)
    • 如果存在，证明存在有环。返回当前环

# PHP

  • 方案：记录指针

    • PHP使用数组记录，查询复杂度为O(1)
    function hasCycle($head)
{
    $cur         = $head;
    $nodeHistory = [];

    while ($cur != null) {
        if (in_array($cur, $nodeHistory)) {
            return $cur;
        } else {
            array_push($nodeHistory, $cur);
        }

        $cur = $cur->next;
    }
    return null;
}

# Python

  • 方案：记录指针

    • Python使用set记录，查询复杂度为O(1)
    def hasCycle(self,head):
    nodeRecords = set() #节点走过的记录
    currentNode = head  #当前节点

    while currentNode:
        if currentNode in nodeRecords:
            return currentNode
        else:
            nodeRecords.add(currentNode)
            currentNode = currentNode.next
    return None
