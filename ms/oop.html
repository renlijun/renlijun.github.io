# 抽象类

TIP

  • 抽象类是一种不能被实例化的类。只能作为其他类的父类来使用。
  • 抽象类是通过关键字abstract 来声明的。
  • 抽象类与普通类相似，都包含成员变量和成员方法。
  • 抽象类中，如果有抽象方法，子类必须实现。且抽象方法不可以有方法体
  • 如果一个类中有抽象方法，那必须是抽象类

不可被实例化 abstract 存在抽象方法，子类必须实现 抽象方法不可以有方法体

abstract class AbstractClass
{
 // 强制要求子类定义这些方法
    abstract protected function getValue();
    abstract protected function prefixValue($prefix);

    // 普通方法（非抽象方法）
    public function printOut() {
        print $this->getValue() . PHP_EOL;
    }
}

class ConcreteClass1 extends AbstractClass
{
    protected function getValue() {
        return "ConcreteClass1";
    }

    public function prefixValue($prefix) {
        return "{$prefix}ConcreteClass1";
    }
}

$class1 = new ConcreteClass1;
$class1->printOut();
echo $class1->prefixValue('FOO_') . PHP_EOL;

# 接口

TIP

  • 接口不可被实例化
  • 通过关键字 interface 来声明的
  • 抽象类不可以有属性
  • 抽象类中的方法，子类用implements实现，接口方法必须被子类实现。

不可被实例化 interface 不可有方法体 不可有属性 接口方法子类必须实现

interface PersonInterface{

    const SECRET = 1;

    /* every person should be able to age */
    public function doAge();
}

class Joe implements PersonInterface {
    public function doAge($say) {
       echo "In an Austrailian accent, Joe says: $say".PHP_EOL;
    }
}

# __call

TIP

在对象中调用一个不可访问方法时，__call() 会被调用。

class Demo 
{
    public function __call($name, $arguments) 
    {
        // 注意: $name 的值区分大小写
        echo "Calling object method '$name' "
             . implode(', ', $arguments). "\n";
    }
}
$demo = new Demo();
$demo->runTest('in object context');

# __isset

TIP

当对不可访问属性调用 isset() 或 empty() 时，__isset() 会被调用。

isset empty

class Demo 
{
    public function __isset($name) 
    {
        echo "Is '$name' set?\n";
        return isset($this->data[$name]);
    }
}
$demo = new Demo();
var_dump(isset($demo->age));

# __toString

TIP

打印一个对象时被调用

  • 此方法必须返回一个字符串，否则将发出一条 E_RECOVERABLE_ERROR 级别的致命错误。
  • 不能在 __toString() 方法中抛出异常。这么做会导致致命错误。
class Demo 
{
    public function __toString() {
        return '__toString';
    }
}

$demo = new Demo();
echo $demo;
